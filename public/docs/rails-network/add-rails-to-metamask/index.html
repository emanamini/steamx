    Adding Rails Network to MetaMask #

    On smartphones:

    1. Open MetaMask and tap on the network list at the top of the app.
    2. Press Add Network.
    3. Select Custom Network tab.

    1. In this dialog, use the following variables:
      • Network Name: Rails Network
      • RPC URL: https://mainnet.steamexchange.io
      • Currency Symbol: STEAMX
      • Chain ID: 6278
      • Explorer: https://explore.steamexchange.io
    2. Then click on Add to have our network in your Network list.

    On extension:

    1. Open MetaMask extension and tap on the network list at the top-left of the app.
    2. Press Add Network.

    1. Scroll down and select “Add a network manually” link.

    1. In this dialog, use the following variables:

      • Network Name: Rails Network
      • RPC URL: https://mainnet.steamexchange.io
      • Currency Symbol: STEAMX
      • Chain ID: 6278
      • Explorer: https://explore.steamexchange.io

    2. Then click on “Save” to have our network in your Network list.
